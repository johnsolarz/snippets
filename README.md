Install
-------

1. Navigate to `cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/User/`
2. [Download](https://github.com/johnsolarz/snippets/zipball/master) or `git clone git@github.com:johnsolarz/snippets.git`
3. Burninate
